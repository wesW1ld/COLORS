The COLORS application is a website where the user can login and add colors to their database and search for those
colors later, even after logging out or closing the tab.

The technologies used are the LAMP stack: Linux, Apache, MySQL, PHP

To set up, create a .env file in the LAMPAPI folder with your database login info
and change the URL at the top of the JavaScript files to your URL
